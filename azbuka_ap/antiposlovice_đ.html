ЂАВО ДОШАО ПО СВОЈЕ

Ђаво дошао по своје! Проверава да ли је враг однео шалу.
Ђаво више не мора да долази по своје. Јавни извршитељ је сасвим довољан!
Ђаво је непредвидив. Нико не зна када ће доћи по своје!
Ђаво не ферма ни цара, ни ћесара! Кад год хоће, долази по своје!
Ђаво није рекеташ, али редовно долази по своје!
Ђаво без пардона долази по своје! Као да је с нама овце чувао.
Друштвена својина је била свачија и ничија, док ђаво није дошао по своје!
Све је отишло на добош! Ђаво је годинама долазио по своје!
Ђаво је и 1999. године долазио по своје!
Где НАТО долази по своје, ту ђаво нема шта да тражи.

ЂАВО НЕ ОРЕ И НЕ КОПА (ЂАВО НЕ СПАВА)

Ђаво не оре и не копа него смишља генетски модификовану храну!
Ђаво не оре и не копа него смишља коме ће пре запржити чорбу!
Ђаво не оре и не копа него с лакомисленима тикве сади.
Ђаво не оре и не копа него свом снагом ради испод жита.
Ђаво не оре и не копа него по цео дан одваја кукољ од жита.
Ђаво не спава него проверава да ли је враг однео шалу!
Ђаво не спава него с времена на време долази по своје!
Ђаво не спава него анђелу јаму копа!
Ђаво не спава него смишља коме ће пре забити нож у леђа.
Ђаво не спава. Нису му сви врагови на броју!
